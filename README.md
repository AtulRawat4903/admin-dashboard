# Admin Dashboard

A responsive admin dashboard built as part of The Odin Project.

## Preview

![Dashboard Screenshot](images/preview.png)

## Features

- CSS Grid layout
- Flexbox components
- Sidebar navigation
- Dashboard cards
- Announcements section
- Trending section
- Responsive layout
- Hover animations
- Transitions

## Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox
